# 🐾 반려견기질검사 반려견 기질 및 성격 진단 검사

동물 행동학 이론과 반려동물 심리학적 지표를 기반으로, 우리 반려견의 **타고난 내재적 기질**과 **후천적 조율 성향**을 입체적으로 분석해주는 반응형 웹 애플리케이션입니다. 

기존 인간 중심의 심리검사를 단순히 차용한 것이 아니라, 실제 수의 행동학에서 다루는 요소들을 바탕으로 독자적인 진단 알고리즘을 설계했습니다.

![반려견기질검사 썸네일](https://images.unsplash.com/photo-1543466835-00a7907e9de1?auto=format&fit=crop&q=80&w=1200)

---

## ✨ 핵심 기능 (Features)

- **20문항 행동학 체크리스트**: 5점 리커트(Likert) 척도를 통해 정확도 높은 행동 데이터 수집
- **이원화된 지표 분석 알고리즘**:
  - **본능적 기질 4축**: 탐험성, 안전성, 교감성, 집중성
  - **후천적 성향 3축**: 독립성, 사교성, 유연성
- **Chart.js 레이더 차트 시각화**: 기질과 성향을 두 개의 다이어그램으로 분리하여 직관적으로 제공
- **맞춤형 양육 리포트**: 연산된 결과 조합에 따른 전문적이고 유머러스한 맞춤형 양육 솔루션 출력
- **고성능 CSR (Client-Side Rendering)**: 순수 HTML/CSS/JS 기반으로 API 호출 비용이 0원이며 매우 빠름
- **소셜 공유 기능**: 카카오톡 SDK 연동 및 클립보드 복사(Fallback) 완벽 지원
- **SEO & 애드센스 최적화**: 시맨틱 웹 구조, JSON-LD 구조화 데이터, 광고 플레이스홀더 내장

## 🧬 행동학적 기반 지표 설계 구조

이 프로젝트는 기존 TCI(기질 및 성격 검사) 등 특정 상용 검사 도구의 저작권과 상표권을 전면 우회하기 위해, 순수 수의 행동학 및 동물 학습 이론에 기반한 독창적인 용어와 구조로 재매핑되었습니다.

### 1. 내재적 기질 (Inherent Traits)
유전자와 신경전달물질에 의해 결정되는 선천적인 반응 체계입니다.
- **탐험성 (E)**: 새로운 자극에 다가서는 호기심 드라이브 (도파민 관련)
- **안전성 (S)**: 위험/낯선 자극에 대한 회피와 신중함 (세로토닌 관련)
- **교감성 (M)**: 보호자와의 유대 및 사회적 친화력 (옥시토신 관련)
- **집중성 (F)**: 즉각적인 보상 없이도 목표를 지속하는 인내력

### 2. 후천적 조율 성향 (Acquired Traits)
양육 환경, 사회화 교육, 보호자의 규칙에 의해 형성되는 성격입니다.
- **독립성 (I)**: 보호자 부재 시 스트레스를 자율적으로 조절하는 능력
- **사교성 (O)**: 낯선 사람이나 타견과의 부드러운 상호작용 및 친화 능력
- **유연성 (X)**: 환경 변화 및 낯선 상황에 대한 회복탄력성

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Library**: [Chart.js](https://www.chartjs.org/) (데이터 시각화)
- **API**: [Kakao JavaScript SDK](https://developers.kakao.com/) (공유 기능)
- **Fonts**: Noto Sans KR, Outfit (Google Fonts)

## 🚀 설치 및 실행 방법

이 프로젝트는 별도의 빌드 과정이나 서버 프레임워크가 필요 없는 정적(Static) 파일로 구성되어 있습니다.

### 1. 로컬 환경에서 실행
```bash
# 저장소 클론
git clone https://github.com/hyunjuncho800/mung-bti.git

# 폴더 이동
cd mung-bti

# index.html 파일을 브라우저에서 바로 열어서 실행
# (VS Code의 Live Server 익스텐션을 사용하면 더욱 편리합니다.)
```

### 2. 카카오톡 공유 기능 활성화 (필수)
`script.js` 파일 상단에 있는 카카오 JavaScript 앱 키를 본인의 키로 교체해야 정상적으로 작동합니다.
```javascript
// script.js 내에서 아래 부분을 찾아 자신의 APP KEY로 변경하세요.
Kakao.init('YOUR_KAKAO_APP_KEY_HERE'); 
```

### 3. 배포 가이드
정적 호스팅 서비스를 통해 즉각적으로 무료 배포가 가능합니다.
- **GitHub Pages**: 저장소의 `Settings` > `Pages`에서 `main` 브랜치를 루트로 설정
- **Vercel / Netlify**: 저장소 연동 후 배포 설정 없이 즉시 배포 가능
